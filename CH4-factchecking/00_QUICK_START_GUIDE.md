# 🚀 給開發者的快速入門指南

## 你拿到這個工作包後該怎麼做？

### 步驟 1：開啟 Cowork（5 分鐘）
1. 確認你有 Claude Max、Pro、Team 或 Enterprise 訂閱
2. 開啟 Claude Desktop 應用程式
3. 點選頂部的「Cowork」標籤（在 Chat 旁邊）
4. 點選「新增工作資料夾」，選擇本工作包的解壓位置

### 步驟 2：給 Cowork 第一個指令（複製貼上即可）

```
你好，我要開發一個教育互動遊戲「AI 真相大搜查」。
請先閱讀資料夾中的 README.md，了解整個專案。
然後依序閱讀 01_遊戲規格書.md、02_內容生產清單.md、
03_程式架構說明.md。

讀完後請告訴我：
1. 你對這個專案的理解
2. 建議的開發順序
3. 我們從哪一步開始最好

請等我確認後再開始實作。
```

### 步驟 3：跟著 Cowork 一步步做

Cowork 會：
1. 讀完所有規格文件
2. 提出開發計畫
3. 開始建立程式碼檔案
4. 每完成一個階段就詢問你

你要做的：
- ✅ 確認每個階段的成果
- ✅ 提供圖片素材（自己用 AI 生成）
- ✅ 錄製或購買配音
- ✅ 測試遊戲是否符合預期

## 工作流程建議

### 第 1 週：規格確認 + 內容產出
**Cowork 做的事**：
- 閱讀所有規格文件
- 細化 3 個情境的腳本
- 撰寫所有 UI 文字
- 建立程式架構（空殼）

**你做的事**：
- 確認 Cowork 的理解正確
- 開始用 Midjourney 生成 3 張主圖
- 在 Unsplash 找真實參考圖

### 第 2 週：程式核心 + 素材整合
**Cowork 做的事**：
- 撰寫 HTML 主架構
- 撰寫 CSS 樣式
- 撰寫 JavaScript 核心邏輯（前 3 個畫面）

**你做的事**：
- 用 Figma 標註破綻 hotspot 座標
- 完成所有圖片素材
- 開始錄製配音

### 第 3 週：完整功能實作
**Cowork 做的事**：
- 完成 3 個關卡的完整邏輯
- 實作結算頁與證書下載
- 整合所有圖片和音效

**你做的事**：
- 將圖片與音效放入正確資料夾
- 進行第一輪測試
- 記錄發現的問題

### 第 4 週：測試與打磨
**Cowork 做的事**：
- 修復發現的問題
- 優化動畫效果
- 處理邊界情況

**你做的事**：
- 完整測試清單檢查
- 找 1-2 位老師試用
- 找 3-5 位學生試玩

## 跟 Cowork 對話的小技巧

### ✅ 好的指令
```
請開始實作開始頁，依照 01_遊戲規格書.md 第 1 節的規格。
完成後請讓我看看，我們再進入下一頁。
```

### ❌ 不夠好的指令
```
做開始頁。
```
（太籠統，Cowork 不知道要做到什麼程度就停）

### ✅ 好的回饋
```
我看了開始頁，有 2 個地方要調整：
1. 標題字太小，請放大到 36px
2. 開始按鈕顏色太淡，請改為更鮮豔的橙色 #FF8C42
其他部分都很好，調整完就可以繼續。
```

### ❌ 不夠好的回饋
```
不太好看，再改一下。
```

## 常見問題

### Q: Cowork 寫的程式有 bug 怎麼辦？
A: 把錯誤訊息或畫面截圖貼給 Cowork，請它修復。它能讀取自己寫的程式碼，並進行除錯。

### Q: 我不會用 Photoshop 標 hotspot 座標怎麼辦？
A: 用免費的 Figma：
1. 上傳圖片到 Figma
2. 用矩形工具畫一個框
3. 查看右側「位置」面板的 X、Y 座標
4. 把座標告訴 Cowork

### Q: 我沒有 Midjourney 訂閱怎麼辦？
A: 用免費替代品：
- **Bing Image Creator**（免費，需 Microsoft 帳號）
- **Google Imagen Studio**（部分免費）
- **Leonardo.AI**（每天免費額度）

### Q: 配音找不到人錄怎麼辦？
A: 替代方案：
- 用 **ElevenLabs**（AI 配音，有免費額度）
- 用 **Google TTS**（完全免費）
- 自己用手機錄音（孩子們不在意是否專業）

### Q: 完成後怎麼給學校用？
A: 三種方式：
1. **USB 隨身碟**：複製整個 game/ 資料夾，雙擊 index.html 即可玩
2. **GitHub Pages**：免費部署到網路上，給連結
3. **學校內網**：請 IT 人員上傳到學校伺服器

## 我需要花多少錢？

### 最低成本方案（接近 $0）
- Cowork 訂閱：你已經有了
- 圖片：用 Bing Image Creator（免費）
- 配音：用 ElevenLabs 免費版
- 音效：用 freesound.org 免費資源
- 部署：用 GitHub Pages（免費）

**總計：$0 - $10**

### 推薦成本方案（$50-100）
- Midjourney 一個月：$10
- 音效素材包：$20-30
- ElevenLabs 付費：$5/月
- 其他：圖庫購買 $20

**總計：$50-100**

### 完整外包方案（$300-600）
- AI 圖片：$30-60
- 真人配音：$200-300
- 設計師後製：$100-200

**總計：$330-660**
