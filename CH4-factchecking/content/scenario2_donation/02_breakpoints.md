# 情境 2：校園捐款溫馨騙局 — 第二關破綻說明

## 主圖
「某企業捐贈 100 臺平板給偏鄉小學」的 AI 生成新聞圖

圖片內容：一群小學生開心地拿著平板，背後有「企業 LOGO」，畫面溫馨但細看有問題。

## 主圖上的 3 個破綻

### 破綻 B2-01：企業 LOGO 模糊或變形
- **位置**：圖片背景看板（座標約 600, 100，範圍 150x100）
- **可見特徵**：企業標誌看起來像知名品牌但字母錯亂或變形
- **說明卡內容**：
  ```
  🔍 破綻 1：企業 LOGO 不對勁
  
  仔細看背景看板上的企業標誌，
  你發現什麼了嗎？
  
  字母好像不對、顏色微妙不對、
  形狀也有點怪。
  
  AI 很難精確複製真實的品牌標誌，
  詐騙者用這種方法假冒大企業。
  
  💡 記住：真正的企業捐贈會有清楚正確的品牌標誌！
  ```

### 破綻 B2-02：學生與平板的互動不自然
- **位置**：學生的手部特寫（座標約 300, 400，範圍 120x120）
- **可見特徵**：握平板的手指數量不對、或平板螢幕角度不合常理
- **說明卡內容**：
  ```
  🔍 破綻 2：手部細節異常
  
  數一數圖片中學生的手指，
  或看看他們握平板的方式，
  是不是有點不自然？
  
  AI 生成圖片最常出錯的地方就是
  手指（常常多 1 根或少 1 根）、
  或者物品和身體的接觸方式。
  
  💡 記住：手部和細節是 AI 圖片的破綻！
  ```

### 破綻 B2-03：背景環境融合痕跡
- **位置**：教室背景某處（座標約 100, 300，範圍 200x150）
- **可見特徵**：教室桌椅、地板、牆壁的交界處模糊、物品邊緣融化
- **說明卡內容**：
  ```
  🔍 破綻 3：背景融合痕跡
  
  看看教室的桌子、椅子、地板的交界處，
  是不是有「融化」、「模糊」的感覺？
  
  AI 在處理多個物體的邊界時容易出錯，
  讓物品看起來像被「P」上去的。
  
  💡 記住：真實照片的物體邊界清晰銳利！
  ```

## 教學備註

這 3 個破綻組合特別有教育意義：
1. **LOGO 偽造** → 提醒學生品牌驗證
2. **手指變形** → AI 圖片最經典的破綻
3. **背景融合** → 後製合成的痕跡

讓孩子建立「看圖三檢查」的習慣：
✓ 檢查 LOGO/標誌
✓ 檢查手部和細節
✓ 檢查邊界銜接
