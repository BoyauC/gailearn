# 情境 1：颱風停課事件 — 第三關角色對白

## 第三關情境描述

「吱吱說：我們找到了 3 個破綻：
- 沒有官方標誌
- 文字排版不規則  
- 時間戳記可疑

這個停課公告應該問誰最能確認真假？」

## 3 個角色選項

### 角色 A：班導師 / 校長 ✅ 正確答案

- **頭像**：穿襯衫、戴眼鏡的老師形象，溫和微笑
- **顯示名稱**：「王老師（班導）」
- **回應文字**（完整對白）：
  ```
  哎呀，這個停課通知是假的！
  
  你看，學校的停課通知一定會透過
  「全校廣播」或「校務系統」正式發布，
  不會用 LINE 群組或網路截圖傳遞。
  
  而且，你發現的這 3 個破綻：
  ❌ 沒有教育部官方徽章
  ❌ 文字排版亂七八糟
  ❌ 連日期都寫錯
  
  100% 是 AI 假造的！
  以後看到停課訊息，記得直接到
  「教育部官網」或打電話到學校確認喔！
  ```
- **配音指示**：溫和、堅定、像在教導
- **配音檔名**：`scenario2_voice_correct.mp3`

### 角色 B：LINE 群組同學 ❌ 錯誤選項

- **頭像**：拿著手機驚慌的同學
- **顯示名稱**：「小明（LINE 同學）」
- **回應文字**：
  ```
  天啊！我也有看到這個消息！
  我媽媽群組也在傳！
  
  那明天真的不用上課嗎？
  好棒喔！我要去睡到中午！
  ```
- **教學意義**：同學也只是看到謠言，並沒有確認來源
- **配音指示**：興奮、毫無懷疑
- **配音檔名**：`scenario2_voice_classmate.mp3`

### 角色 C：網路評論者 ❌ 錯誤選項

- **頭像**：戴帽子、看不清臉的網友
- **顯示名稱**：「神秘網友」
- **回應文字**：
  ```
  感覺好像是真的耶...
  最近颱風確實很多。
  
  我認識的人也說可能會停課，
  我們就等等看吧！
  ```
- **教學意義**：網友沒有官方身份，無法確認真假
- **配音指示**：含糊、不確定
- **配音檔名**：`scenario2_voice_netizen.mp3`

## 選擇正確後的劇情推進

玩家選對「班導師」後：
1. 班導師說完整對白
2. 主播吱吱出現：「太棒了！你成功識破了假停課訊息！」
3. 顯示「獲得 🏆 第 3 個獎盃」
4. 進入結算頁

## 選擇錯誤後

玩家選錯（同學或網友）：
1. 該角色說完回應
2. 主播吱吱出現：「嗯...這個答案好像不夠好。再想想，誰最能確認官方公告？」
3. 玩家可重新選擇（不扣時間、不扣分）

## 設計重點

✅ 正確答案的角色（老師）回應要：
- 明確指出「假的」
- 解釋為什麼是假的（呼應 3 個破綻）
- 教導正確的查證管道

✅ 錯誤選項的角色要：
- 不要說「對啊是真的」（避免誤導）
- 表現出「沒能力查證」的特質
- 讓玩家自己感受到「問這個人沒用」
