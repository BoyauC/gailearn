# 情境 2：颱風停課事件 — 第二關破綻說明

## 玩家從第一關獲得的圖片
訊息 3 或 訊息 4 的截圖（玩家點擊停下的其中一張）

預設使用「訊息 3：超級颱風來襲全臺停課」作為主圖

## 主圖上的 3 個破綻

### 破綻 B2-01：沒有官方標誌
- **位置**：圖片左上角（座標約 50, 50，範圍 100x80）
- **可見特徵**：應該有「教育部」或「氣象署」徽章的地方是空白或模糊
- **說明卡內容**：
  ```
  🔍 破綻 1：沒有官方標誌
  
  真正的官方新聞會有清楚的「教育部」、
  「氣象署」徽章或浮水印。
  
  AI 生成的假新聞常常會漏掉或模糊化這些細節，
  因為 AI 很難精確複製複雜的徽章圖案。
  
  💡 記住：官方訊息一定有官方標誌！
  ```

### 破綻 B2-02：文字排版不規則
- **位置**：圖片中段（座標約 400, 300，範圍 200x150）
- **可見特徵**：字體大小不一致、行距異常、有的字粗有的字細
- **說明卡內容**：
  ```
  🔍 破綻 2：文字排版不規則
  
  正式公告的字體大小和排版會非常統一，
  因為是用專業排版軟體製作的。
  
  AI 生成的假新聞可能會出現字體大小忽大忽小、
  行距異常、甚至有些字看起來「歪歪的」。
  
  💡 記住：正式公告排版一定整齊！
  ```

### 破綻 B2-03：時間戳記可疑
- **位置**：圖片右下角（座標約 600, 500，範圍 150x80）
- **可見特徵**：發布時間顯示「2025/13/45」這種不存在的日期，或時間格式錯誤
- **說明卡內容**：
  ```
  🔍 破綻 3：時間戳記可疑
  
  仔細看發布時間或日期，
  你發現有什麼不對勁嗎？
  
  AI 生成的假新聞常常在小細節上出錯，
  例如：不存在的日期（13 月 45 日）、
  邏輯不通的時間（25:99）、
  或者格式錯亂的時間戳。
  
  💡 記住：細節決定真假！
  ```

### 視覺提示
- 玩家點擊空白處：什麼都不發生（不扣分）
- 玩家點擊正確位置：該位置出現紅色圓圈標記 + 彈出說明卡
