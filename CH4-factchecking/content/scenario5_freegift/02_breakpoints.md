# 情境 5：免費禮物騙局 — 第二關破綻說明

## 主圖
「免費 iPhone 15」的詐騙廣告 AI 圖片，包含禮物盒、iPhone、誇張文字

## 3 個破綻

### 破綻 B5-01：禮物盒的質感不真實
- **位置**：禮物盒中央（座標約 400, 300，範圍 150x150）
- **可見特徵**：禮物盒紋理過於完美、緞帶反光不自然、陰影方向矛盾
- **說明卡內容**：
  ```
  🔍 破綻 1：禮物盒太完美了
  
  仔細看這個禮物盒，
  是不是覺得「太完美」、「太閃亮」？
  
  真實照片中的物品會有：
  - 自然的瑕疵
  - 合理的陰影
  - 正確的反光方向
  
  AI 生成的物品常常「太理想化」，
  反而顯得虛假！
  
  💡 記住：太完美的東西要懷疑！
  ```

### 破綻 B5-02：iPhone 螢幕內容錯誤
- **位置**：iPhone 螢幕（座標約 300, 450，範圍 80x140）
- **可見特徵**：螢幕上的 App 圖示變形、字體錯亂、邊界模糊
- **說明卡內容**：
  ```
  🔍 破綻 2：iPhone 螢幕怪怪的
  
  看看 iPhone 螢幕上的內容：
  - App 圖示是不是變形了？
  - 文字是不是看不清楚？
  - 螢幕邊界是不是模糊？
  
  AI 很難精確繪製手機螢幕的細節，
  尤其是小圖示和文字。
  
  💡 記住：螢幕細節是 AI 圖的弱點！
  ```

### 破綻 B5-03：背景與物體的銜接不自然
- **位置**：物體底部與背景的交界處（座標約 400, 600，範圍 250x100）
- **可見特徵**：禮物盒底部與背景有「漂浮感」、陰影不對、邊緣融化
- **說明卡內容**：
  ```
  🔍 破綻 3：物體像「漂」在空中
  
  看看禮物盒和 iPhone 的底部，
  它們和背景的接觸點看起來如何？
  
  真實照片中，物體會有：
  - 自然的陰影
  - 與表面的接觸感
  - 清晰的邊緣
  
  AI 圖常常讓物品看起來「漂浮」，
  或邊緣模糊融化。
  
  💡 記住：注意物體與背景的關係！
  ```
