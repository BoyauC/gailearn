# 情境 3：免費 iPhone 廣告 — 第二關破綻說明

## 主圖
第二關隨機從 `s3_msg2.webp`（恭喜！免費獲得 iPhone 17！）或 `s3_msg4.webp`（按讚分享免費送！）抽 1 張為主圖；每張各 3 個破綻，共 6 個。

座標來自 `assets/images/scenarios/s3/marked/s3_msg{2,4}-marked.webp` 用綠色圈標示後以 PIL + scipy 自動偵測得出。

## 主圖 A：s3_msg2（限時 24 小時 iPhone 詐騙）

### 破綻 B3-01：活動限時時間緊迫
- **位置（百分比）**：(38.79, 21.09)，容錯 10.5
- **可見特徵**：上方倒數計時 + 「限時 24 小時！」字樣
- **說明卡內容**：
  ```
  🔍 破綻 1：活動限時時間緊迫

  「限時 24 小時！」這類倒數時間是詐騙常見手法，
  目的是逼你不假思索快點點下去。
  真實活動的時間會清楚標示主辦單位與正式期限，
  不會用施壓口吻。

  💡 記住：越急著要你做決定 = 越可能是詐騙！
  ```

### 破綻 B3-02：不需支付費用
- **位置（百分比）**：(65.29, 20.93)，容錯 14.5
- **可見特徵**：醒目按鈕「不需支付任何費用」
- **說明卡內容**：
  ```
  🔍 破綻 2：不需支付費用

  強調「不需支付任何費用」反而是警訊。
  正常的活動不會特別強調這件事，
  因為對方在用「免費」吸引你忽略風險。

  💡 記住：刻意強調免費的，多半在偷你的個資或時間！
  ```

### 破綻 B3-03：沒有品牌或廠商標示，只有 QR Code
- **位置（百分比）**：(84.04, 81.08)，容錯 13.2
- **可見特徵**：右下角只有 QR Code + 「馬上領取」，沒有公司名、聯絡資訊、活動細則
- **說明卡內容**：
  ```
  🔍 破綻 3：沒有品牌或廠商標示

  只附一個 QR Code 跟「馬上領取」，
  沒有任何公司名稱、聯絡方式、活動細則。
  正常廠商活動會清楚標出誰主辦、怎麼聯絡。

  💡 記住：QR Code 是詐騙最愛用的入口，不要隨便掃！
  ```

## 主圖 B：s3_msg4（按讚分享免費送）

### 破綻 B3-04：『按讚分享免費送』典型話術
- **位置（百分比）**：(50.00, 14.00)，容錯 20.0
- **可見特徵**：標題大字「按讚分享免費送！」
- **說明卡內容**：
  ```
  🔍 破綻 4：『按讚分享免費送』典型話術

  要求你按讚／分享才能領取的訊息，
  幾乎都是用來擴散假廣告的詐騙手法。
  你幫忙散播 → 對方騙到更多人。

  💡 記住：要你幫忙散播的「好康」十之八九有問題！
  ```

### 破綻 B3-05：數量有限營造限量
- **位置（百分比）**：(87.42, 73.83)，容錯 12.25
- **可見特徵**：「數量有限，送完為止」標語
- **說明卡內容**：
  ```
  🔍 破綻 5：數量有限營造限量

  「數量有限，送完為止」是製造緊張感的話術，
  逼你立刻行動、來不及思考。
  真實的好康不會用這種催促語氣。

  💡 記住：強調限量 + 倒數 = 詐騙手法！
  ```

### 破綻 B3-06：完全免費不合邏輯
- **位置（百分比）**：(88.62, 25.39)，容錯 11.29
- **可見特徵**：右上角「完全免費，0 元帶回家」標語
- **說明卡內容**：
  ```
  🔍 破綻 6：完全免費不合邏輯

  「完全免費，0 元帶回家」— 想想看，真有那麼好的事嗎？
  豪華禮物的成本誰要付？
  背後一定有想從你身上拿走的東西（個資、點數、信用卡）。

  💡 記住：天下沒有白吃的午餐！
  ```
